# MABS (0.1b2)

### Installation:
- Step 1: Nothing! This script does not need installing it is 100% portable!!!
- If you would like to manually build:
`
gcc .\src\main.c -o MABS
`

---

### About MABS
- MABS is an open source improvment tool for winslop
- MABS allows for a user to install programs onto their computer with [Scoop](https://scoop.sh/)
- MABS is written in c for minimum overhead (commands are run via powershell which creates huge slowness)
- MABS is also written in a way that makes it easy for anyone to add they own processes to the script :)

### Why should I use MABS
- MABS allows you to use a system how you want to
- It give freedom to do what you want with your hardware
- Can bypass any silly restrictions
